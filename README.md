# Leaf-Hunter

## Purpose

Memory game using React. Can you beat the timer?

## Getting Started

Click on a leaf to get a point. Avoid clicking the same leaf twice!

Click [here](https://jonathan-white.github.io/Leaf-Hunter/) to start the game!
